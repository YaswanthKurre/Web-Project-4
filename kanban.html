Arena Realty

Scrum Methodology

We followed Scrum principles and practices for development of the Arena Realty property website.

Project Vision

Create a user-friendly platform to connect home buyers and sellers.

Team

  • Product Owner - Defines product features
  • Scrum Master - Manages process and timelines
  • Developers - Build the product

Epics

  • User registration and login
  • Seller property listings
  • Buyer search and management
  • Admin analytics

Sprints

We changed normal 2 week cycles into 1 week cycle for sprint planning to deliver working software increments.

  • Sprint 1
    • User registration and login forms
    • Dashboard shell pages
    • Seller property listing and management
  • Sprint 2
    • Database integration
    • Buyer search
    • Admin analytics and reporting

Daily standups were used to track progress, blockers, next steps.

Retrospectives at the end of each sprint to improve our process.
